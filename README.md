# pfSense Firewall VM

- Primary firewall and router with NAT and DHCP
- LAN/WAN network configuration
- Basic firewall rules for allowing and restricting traffic
- Traffic monitoring and network event logging

# Windows VM (Workstation)

- Connected to the LAN behind the pfSense firewall
- Test client for firewall rule validation
- Used for traffic analysis (Wireshark) and security policy testing
